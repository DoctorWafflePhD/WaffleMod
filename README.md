# WaffleMod
**WaffleMod** is an assorted content mod for Balatro with varying degrees of thematic cohesion, aiming to add to the vanilla experience while also adding new concepts to play with.
Currently, **WaffleMod** adds...
- 2 new decks
    - 1 new sleeve, if DeckSleeves is installed
- 18 Jokers (8 Common, 4 Uncommon, 3 Rare, 3 Legendary)
- 1 new Tarot card
- 1 new Enhancement
- 1 new Edition
- 2 new Seals
- 2 new Spectral cards
- 3 new Boss Blinds
- 2 new deck skins, one for Hearts, one for Diamonds
There are additional features which are not enabled by default (as they are unfinished). As there is currently no in-mod config menu, enabling these will have to be done manually within AppData\Roaming\Balatro\Config\wafflemod.jkr.
These experimental features include...
- 3 Arcade Cabinets, a new consumable type
    - Arcade Cabinets cost money to activate, but are not consumed upon use, and additionally provide passive effects while held.
- Boss Jokers, a new rarity of Joker that have a 1 in 8 chance to drop from a defeated Boss Blind, provided you have an empty Joker slot.
    - Only 7/31 have been created so far, including...
        - The Club, Goad, Head, and Window (suit bosses)
        - The Needle
        - The Ox
        - The Serpent
- Reverse Tarot cards, which have a rare chance to replace Tarot Cards in the shop and in Arcana Packs, offering new effects.
